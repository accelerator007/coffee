# District 7 — التصميم الجديد · What changed & what's missing

أعيد تصميم التطبيق بالكامل على هوية **District 7** (مونوكروم أسود على أبيض، حواف رفيعة،
خطوط Playfair Display + Amiri للعناوين و Inter + Almarai للنصوص). الباك-إند الحالي
(Supabase) ظل كما هو، والتصميم الجديد مربوط به مباشرة.

---

## ✅ مربوط بالباك-إند فعلياً (بيانات حقيقية)
- **تسجيل الدخول** (عميل بالهاتف / موظف) — كما هو، بشكل جديد.
- **لوحة العميل**: الباقة + الأيام المتبقية + أكواب اليوم + رمز QR (QR حقيقي قابل للمسح).
- **مسح الموظف** (QR / NFC) وتسجيل الكوب — نفس المنطق، بدون أي تغيير في الوظيفة.
- **لوحة الأدمن**: مؤشرات (عملاء / فعّالة / استرداد / **الإيرادات OMR**)، **رسم الأسبوع**،
  جدول الباقات والعملاء — مع **شارات الفئة** (ذهبية/فضية/برونزية).
- إدارة الباقات والعملاء والاشتراكات والحسابات.

## ⚙️ مطلوب منك لتشغيل كل شيء
1. **شغّل الـ migration الجديد في Supabase** (SQL Editor → New query):
   - الصق محتوى `supabase/migrations/0004_d7_design.sql` واضغط Run.
   - (أو الصق `supabase/migrations/full_schema.sql` كامل على مشروع جديد — محدّث ويتضمن 0004.)
   - يضيف: عمود `tier` للباقات + الإيرادات في مؤشرات الأدمن + دالة رسم آخر 7 أيام
     + إرجاع `tier` في دوال العميل والأدمن.
2. **حدّد فئة كل باقة** من صفحة **الأدمن → الباقات** (Tier: ذهبية/فضية/برونزية) حتى تظهر الشارات.
3. **ملف `.env.local`** (مطلوب لتشغيل الموقع محلياً أو على Netlify):
   ```
   NEXT_PUBLIC_SUPABASE_URL=...
   NEXT_PUBLIC_SUPABASE_ANON_KEY=...
   SUPABASE_SERVICE_ROLE_KEY=...   # لإدارة العملاء (موجود مسبقاً في مشروعك)
   ```

## 🎨 تزييني / غير مربوط (مقصود)
- **شريط «من حسابنا» (Instagram feed)**: بلاطات ثابتة فقط — الربط الحقيقي يحتاج
  تكامل Instagram Graph API + توكن (خارج النطاق الحالي).
- **أسهم النِسب في المؤشرات (▲ ▼ %)**: غير مفعّلة لعدم وجود بيانات فترة سابقة للمقارنة.
  (المكوّن `KPICard` يدعم خاصية `delta` متى توفّرت بيانات تاريخية.)

## 🔎 كيف تتأكد أن كل شيء يعمل
```bash
npm install
npm run build      # ✅ يبني بنجاح (تم التحقق)
npm run lint       # ✅ نظيف
npm run dev        # للتشغيل محلياً (بعد إضافة .env.local)
```
> الخطوط تُحمّل وتُستضاف ذاتياً عبر `next/font/google` (Playfair Display · Amiri · Inter · Almarai).
